# MythTV

## Install Ubuntu

- Install Ubuntu 18.04 LTS.
  - Minimal Installation
  - Install 3rd party codecs, etc.
  - Erase disk and install
    - Don't encrypt the disk; will use encrypted home directories.
    - Use LVM (for snapshots, etc.)
- Install updates; reboot.
- Use ukuu to install a 4.20 kernel
  - See https://itsfoss.com/upgrade-linux-kernel-ubuntu/
  - `sudo ukuu --install v4.20.17`
  - Better support for Intel graphics firmware blob in the NUC, apparently.
