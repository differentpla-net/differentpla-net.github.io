## Installing Elixir

```
pi@raspberrypi:~ $ cat /etc/apt/sources.list.d/erlang-solutions.list
deb https://packages.erlang-solutions.com/debian stretch contrib
```

```
sudo apt-get update && sudo apt-get upgrade && sudo apt-get install -y elixir
```

This installs Elixir 1.6.1 and Erlang/OTP 20.1.7
